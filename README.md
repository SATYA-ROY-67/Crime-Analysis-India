📊 District-wise Crime Analysis in India (2001–2012)
📌 Project Overview

This project analyzes district-wise crime data in India from 2001 to 2012 using data science techniques. The focus is on understanding patterns in murder and other IPC crimes.

🎯 Objectives
Analyze crime trends over time
Identify high-crime states and districts
Study relationships between different crime types
Visualize crime patterns using graphs
📂 Dataset
Source: Government crime data (NCRB)
Rows: 9000+
Columns: 30+
Features:
State/UT
District
Year
Murder
Rape
Robbery
Theft
Total IPC Crimes
⚙️ Technologies Used
Python
Pandas
Matplotlib
Seaborn
🔍 Key Analysis
Year-wise murder trend
Top 10 states with highest murders
District-wise crime analysis
Crime distribution
Correlation between crimes
📈 Visualizations
Line Graph (Murder Trend)
Bar Chart (Top States)
Pie Chart (Crime Distribution)
Heatmap (Correlation Matrix)
🧠 Key Insights
Uttar Pradesh has highest murder cases
Crime trends fluctuate but remain high
Theft contributes major portion of crimes
Strong correlation between robbery, theft, and murder
Total IPC crimes show increasing trend
📌 Conclusion

This project demonstrates how data science techniques can be used to analyze real-world crime data and extract meaningful insights for better decision-making.
